
# Calendar-planer

Calendar-planer profides graphical interface and basic planning options: 
- create/delete/finish tasks
- sort tasks by categories
- put deadline dates and time for each tasks
- put the task on the optimal date before deadline


## Features

- Simple interface
- Internet connection independency
- Fullscreen mode
- Lightweight



## Technologies

- C++: algorythms
- Qt: interface
- SQLite: data storage
